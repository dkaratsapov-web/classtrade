# CLASS TRADE

Сайт компании CLASS TRADE — профессиональное оснащение учебных заведений.
Стек: **React + Vite**. Реализована **главная страница** по дизайн-макетам.

## Запуск

```bash
npm install
npm run dev      # дев-сервер
npm run build    # продакшн-сборка в dist/
npm run preview  # предпросмотр сборки
```

База макета — десктоп **1440 px**, контент-контейнер **1240 px** (шапка/подвал/полосы — до 1320). Адаптив проектируется отдельно.

## Структура

- `src/App.jsx` — сборка секций главной (порядок по памятке, раздел 8).
- `src/components/` — секции (Header, Hero, Advantages, Categories, Feature, Spheres, About, Partners, News, Contacts, Footer) и общие элементы (`common.jsx`, `icons/`).
- `src/index.css` — токены (CSS-переменные) и стили по фирменным паттернам CLASS TRADE.
- `public/assets/` — фото-рендеры, используемые на странице.

## Дизайн-источники (`design/`)

- `design/CLASS-TRADE-памятка-для-Claude-Code.md` — практическая шпаргалка по вёрстке (читать первой).
- `design/CLASS-TRADE-design-handoff.md` — полная спецификация: палитра, типографика, компоненты.
- `design/reference/Главная.dc.html` — эталонная вёрстка главной (визуальный ориентир №1).
- `design/reference/assets/` — исходные фото-рендеры.

> Источник истины — PDF-макеты и handoff. Вёрстка должна попиксельно совпадать с макетом; правки стиля — только через согласование. Остальные 17 экранов делаются по этой же системе после согласования главной.
