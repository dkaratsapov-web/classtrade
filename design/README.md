# CLASS TRADE — пакет для вёрстки (handoff для Claude Code)

Это всё, что нужно для вёрстки сайта CLASS TRADE. Раздутый PDF (57 МБ) не нужен — здесь лёгкий пакет с уже извлечёнными данными.

## Что внутри
- `CLASS-TRADE-памятка-для-Claude-Code.md` — **читать первым.** Практическая шпаргалка: токены, фишки стиля с готовым кодом, чек-лист «чего не делать».
- `CLASS-TRADE-design-handoff.md` — полная спецификация: палитра, типографика, компоненты, поэкранная структура.
- `reference/Главная.dc.html` — эталонная вёрстка главной (открывается в браузере, рядом лежит `support.js`). Визуальный ориентир №1.
- `reference/assets/` — реальные фото-рендеры, извлечённые из макета (классы, кабинеты, спортзал, детсад).

## Как этим пользоваться в Claude Code
1. Распакуй этот архив в корень своего git-репозитория (например в папку `design/` или в корень проекта).
2. `git add . && git commit -m "design handoff: главная CLASS TRADE"` — после коммита Claude Code видит все файлы напрямую.
3. В сессии Claude Code дай задачу: «Свёрстай главную по `CLASS-TRADE-памятка-для-Claude-Code.md`, эталон — `reference/Главная.dc.html`, фото в `reference/assets/`».
4. Остальные 17 экранов — по той же системе, после согласования главной.

> Эталон сделан на инлайн-стилях (`reference/Главная.dc.html`). Стек вёрстки выбирай свой (чистый HTML/CSS, React и т.п.) — важно совпадение с макетом, а не способ.
