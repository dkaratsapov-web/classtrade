# CLASS TRADE — дизайн-спецификация для вёрстки

> Сайт компании **CLASS TRADE** — профессиональное оснащение учебных заведений (детсады, школы, доп. образование, медицина, культура, спорт).
> Источник истины — приложенные PDF-макеты (18 экранов). Вёрстка должна **максимально совпадать** с макетами.
> Базовая ширина макета — **1440 px**. Эта спецификация фиксирует глобальную систему; первой реализуется главная страница.

---

## 1. Фирменный стиль — ключевые моменты (на что зафиксировать внимание)

Это НЕ минималистичный «эпловский» стиль. Это **энергичный, плотный, «спортивно-молодёжный» дизайн** со своими фишками. При правках их нельзя терять:

1. **Двухцветные «стикерные» заголовки.** Каждый заголовок секции разбит на 2 слова-плашки: одна на **тёмно-синем** фоне, вторая на **белом** (или наоборот), каждая со скруглением ~8px и **слегка повёрнута** (±2–4°) в разные стороны — эффект наклеенных стикеров. Текст крупный, жирный, ЗАГЛАВНЫМИ. Это главный узнаваемый приём — встречается в каждой секции.
2. **Фирменная «S»-лента.** Логотип — буква **S** в скруглённом синем квадрате. Этот силуэт S «вытекает» из логотипа и **огромной волнистой лентой (синей #036AFF + белой) вьётся сквозь страницу**, связывая секции и перетекая из светлых блоков в тёмные. Это фоновый декоративный слой, не контент.
3. **Точечная текстура.** Светло-голубые блоки покрыты **сеткой мелких точек** (dot-grid) — лёгкий фоновый паттерн, придаёт «технологичность».
4. **Чередование фонов.** Светло-голубой `#EDF9FF` ↔ белый ↔ тёмно-синий `#01254F`. Ритм задаётся сменой фона, а не линиями.
5. **Сквозные синие «бегущие строки».** Полноширинные ярко-синие полосы `#036AFF` с белыми ЗАГЛАВНЫМИ пунктами-преимуществами, разделёнными иконкой-логотипом (квадрат с S).
6. **Круглая кнопка-стрелка.** Рядом с каждой pill-кнопкой («Перейти в каталог», «Подробнее», «Оставить заявку») стоит **отдельная круглая синяя кнопка со стрелкой →**. Парный элемент, повторять везде.
7. **Тёмные фото-плитки категорий.** Карточки направлений — фото помещения с тёмным затемнением, белый ЗАГЛАВНЫЙ заголовок снизу-слева и круглая синяя стрелка снизу-справа.

## 2. Цветовые токены (взяты из PDF)

| Токен | HEX | Применение |
|---|---|---|
| `--blue` (primary) | `#036AFF` | Лента S, кнопки, ссылки, синие полосы, акценты, фичер-блоки |
| `--navy` (dark) | `#01254F` | Шапка, подвал, тёмные секции, тёмные стикер-плашки, основной текст |
| `--navy-2` | `#01245F` | Вариация navy (тени/градации тёмного) |
| `--bg-sky` | `#EDF9FF` | Основной светлый фон страницы (голубоватый) |
| `--white` | `#FFFFFF` | Карточки, светлые плашки, текст на тёмном |
| `--ink` | `#101010` | Тёмный текст на светлом |
| `--placeholder` | `#D9D9D9` | Заглушки LOGO партнёров, фон полей формы |
| `--border` | `#E0E0E0` | Тонкие границы карточек/полей |

Единый акцент — синий `#036AFF`. Второго бренд-цвета нет.

## 3. Типографика

- **Гарнитура:** тяжёлый геометрический гротеск. В макете — плотный жирный санс (близко к **Montserrat / TT Firs / Golos**). Рекомендация для вёрстки: `Montserrat` (или `Golos Text` для кириллицы) — веса 400 / 600 / 700 / 800.
- **Заголовки секций (стикеры):** 700–800, UPPERCASE, межбуквенный 0, крупный кегль (на 1440 ≈ 44–56px), плашки с paddings ~12×24px, radius 8px, поворот ±2–4°.
- **Главный H1 (hero):** 800, UPPERCASE, ~64–72px, цвет navy на светлом.
- **Подзаголовки внутри блоков:** 700, UPPERCASE, ~22–26px.
- **Body:** 400–500, ~16–18px, line-height ~1.5, цвет navy/ink.
- **Мелкий текст / подписи:** ~13–14px, приглушённый.
- Без эмодзи.

## 4. Глобальные компоненты

### Шапка (header), фон `--navy`, высота ~88px
Слева: **логотип** (синий скруглённый квадрат с белой «S» + текст `CLASS TRADE` белым, жирный).
Далее: синяя pill-кнопка **«Каталог»** с иконкой-гамбургером. Затем навигация (белым): `О компании` · `Проекты` · `Блог` · `Контакты`.
Справа: иконка **поиск**, белая контурная pill **«Заказать звонок»**, иконка **сердце** (избранное), иконка **корзина**. Иконки — тонкие линейные, в круглых контурных рамках.

### Кнопки
- **Primary pill:** фон `--blue`, текст белый, радиус pill, рядом — отдельная круглая синяя кнопка со стрелкой →.
- **Ghost pill (на тёмном/светлом):** прозрачная с контуром, текст белый/navy.
- Состояние нажатия: лёгкое `scale(0.97)`.

### «Бегущая строка» (advantage bar)
Полноширинная полоса `--blue`, h≈72px. Внутри 3–4 пункта белым UPPERCASE (напр. «РАБОТА С ГОСЗАКАЗЧИКАМИ», «ШИРОКАЯ ГЕОГРАФИЯ ПОСТАВОК», «ПОЛНЫЙ ПАКЕТ ДОКУМЕНТОВ»), между ними — иконка-логотип (белый квадрат с S).

### Подвал (footer), фон `--navy`
Слева логотип + строка «Весь контент на этом сайте защищён авторским правом». Справа меню (`Каталог · О компании · Проекты · Блог · Контакты`) и строка правовых ссылок (`Договор-оферта · Политика конфиденциальности · Разработка сайта`).

## 5. Структура ГЛАВНОЙ страницы (сверху вниз)

1. **Hero.** Фон navy с точками. H1 «ПРОФЕССИОНАЛЬНОЕ ОСНАЩЕНИЕ» + стикеры «УЧЕБНЫХ ЗАВЕДЕНИЙ». По центру — описание + primary-кнопка «Перейти в каталог». Слева и справа — **два фото-рендера классов** под наклоном, с синей подложкой и фирменной тенью. Снизу — счётчик слайдов `01 / 03` и стрелка-вниз (слайдер).
2. **Преимущества сотрудничества.** Стикер-заголовок. 3 белые карточки: «ШИРОКИЙ АССОРТИМЕНТ ОБОРУДОВАНИЯ», «ВЫСОКИЙ УРОВЕНЬ СЕРВИСА И ПОДДЕРЖКИ», «ОПТОВЫЕ ЦЕНЫ И ГИБКИЕ РЕШЕНИЯ» (иконка + заголовок + текст). Поверх карточек проходит S-лента.
3. **Синяя полоса:** «РАБОТА С ГОСЗАКАЗЧИКАМИ · ШИРОКАЯ ГЕОГРАФИЯ ПОСТАВОК · ПОЛНЫЙ ПАКЕТ ДОКУМЕНТОВ».
4. **«ВСЕ ДЛЯ СОВРЕМЕННОГО ОБРАЗОВАНИЯ».** Сетка тёмных фото-плиток категорий: «ДЕТСКИЙ САД», «ШКОЛА», «СРЕДНЯЯ И СТАРШАЯ ШКОЛА», «ДОПОЛНИТЕЛЬНОЕ ОБРАЗОВАНИЕ», «ИНКЛЮЗИВНОЕ ОБРАЗОВАНИЕ» (фото + затемнение + белый заголовок + круглая стрелка).
5. **«МАГАЗИН УНИКАЛЬНЫХ РЕШЕНИЙ».** Фичер-блок: слева синяя панель с заголовком «КОМПЛЕКСНОЕ ОСНАЩЕНИЕ ДЛЯ ДЕТСКОГО САДА», списком («четыре группы, медицинский кабинет, пищеблок, вспомогательные помещения») и кнопкой «Подробнее»; справа — фото. Через блок проходит белая S-лента. Внизу слайдер `01 / 04`.
6. **«СФЕРЫ ВНЕДРЕНИЯ ОБОРУДОВАНИЯ».** Большой список-аккордеон крупными словами: **МЕДИЦИНА**, **ОБРАЗОВАНИЕ**, **КУЛЬТУРА**, **СПОРТ** — каждое с подписью и фото слева (активный пункт раскрыт фото). Чередование цвета слов navy/blue.
7. **«О КОМПАНИИ CLASS TRADE».** Текст о компании + кнопка «Подробнее», по бокам два фото-рендера, через блок — синяя S-лента.
8. **Синяя полоса:** «10 ЛЕТ НА РЫНКЕ · СНАБДИЛИ 500+ ОБЪЕКТОВ · ИНДИВИДУАЛЬНЫЙ ПОДХОД · ТЕХНИЧЕСКАЯ ПОДДЕРЖКА».
9. **«НАШИ ПАРТНЁРЫ».** Сетка плиток-заглушек «LOGO» (2 ряда), белые карточки.
10. **«НОВОСТИ И СТАТЬИ».** 3 карточки: фото + бейдж «Новый проект» + «НАЗВАНИЕ НОВОСТИ» + описание + кнопка «Подробнее».
11. **«НАШИ КОНТАКТЫ».** Фон navy. Слева белая карточка с формой («Имя», «Телефон», «E-mail», «Комментарий», чекбокс согласия, кнопка «Оставить заявку»). Справа синяя панель с контактами (телефон, e-mail, адрес, e-mail для сотрудничества) и белой S-лентой.
12. **Подвал.**

## 6. Декоративные приёмы (важно для совпадения)
- **S-лента** — рисуется как крупная SVG/clip-path волна, цвет `#036AFF` или белый, z-index под контентом, перетекает между секциями.
- **Dot-grid** — `radial-gradient` точками `#036AFF`/`#cfe0ff` с шагом ~28–32px на светлых секциях.
- **Наклон стикеров** — `transform: rotate()` ±2–4°, плашки слегка перекрывают друг друга.
- **Тень фото** — единая мягкая тень под фото-рендерами (`0 18px 50px rgba(1,37,79,.25)`).
- Скругления: карточки ~20–24px, плашки-стикеры ~8px, кнопки pill, фото-плитки ~16px.

---

## 7. Ассеты
Реальные фото-рендеры извлечены из PDF и лежат в `assets/` (классы, кабинеты, спортзал, детсад). Логотипы партнёров и часть фото — заглушки, заменяются клиентом.

## 8. Замечание по «design system»
К проекту был привязан стиль *Aperture* (минимализм, Apple-like). Он **не соответствует** макетам CLASS TRADE и сознательно **не применяется** — приоритет у приложенных PDF.
