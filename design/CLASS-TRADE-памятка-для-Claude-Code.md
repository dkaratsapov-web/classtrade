# CLASS TRADE — памятка для Claude Code

Краткая практическая шпаргалка по вёрстке. Источник истины — PDF-макеты + файл `CLASS-TRADE-design-handoff.md`. Здесь — то, что **легко упустить**, и готовые паттерны.

База: десктоп **1440 px**, контент-контейнер **1240 px** (шапка/подвал/полосы — до 1320). Адаптив проектируется отдельно, макеты только десктоп.

---

## 0. Главное правило
Вёрстка должна **попиксельно совпадать** с PDF. Не «улучшать», не «причёсывать», не менять отступы/цвета/повороты на свой вкус. Любая инициатива — через согласование.

---

## 1. Токены (вставить как CSS-переменные)

```css
:root{
  --blue:#036AFF;     /* основной акцент: лента, кнопки, ссылки, полосы */
  --navy:#01254F;     /* шапка, подвал, тёмные секции, тёмные плашки, текст */
  --navy-2:#01245F;
  --sky:#EDF9FF;      /* основной светлый фон */
  --white:#FFFFFF;
  --ink:#101010;
  --muted:#5B6B80;    /* вторичный текст */
  --placeholder:#D9D9D9;
  --border:#E0E0E0;

  --r-card:22px; --r-tile:18px; --r-chip:11px; --r-pill:999px;
  --shadow-photo:0 18px 50px rgba(1,37,79,.22);
  --shadow-card:0 14px 40px rgba(1,37,79,.08);
}
```
Единственный акцент — синий. **Второго бренд-цвета нет.** Не вводить градиенты-«украшалки».

---

## 2. Шрифт
- В макете — плотный жирный гротеск. Базовая замена: **Montserrat** (или **Golos Text** для кириллицы), веса 400/500/600/700/**800**.
- Заголовки/плашки — **800, UPPERCASE**. Body — 400–500.
- ⚠️ Уточнить у заказчика фирменный шрифт перед финалом — возможно потребуется замена @font-face.

---

## 3. ❗️ФИШКИ, которые НЕЛЬЗЯ потерять

### 3.1 Двухцветные стикер-заголовки (в каждой секции)
Два слова = две плашки, **по диагонали в две строки** (не встык!), с лёгким поворотом в разные стороны, вторая плашка сдвинута вправо-вниз и лежит поверх первой.

```html
<div class="sticker">
  <span class="chip chip--navy a">СФЕРЫ ВНЕДРЕНИЯ</span>
  <span class="chip chip--navy b">ОБОРУДОВАНИЯ</span>
</div>
```
```css
.sticker{display:flex;flex-direction:column;width:fit-content;margin:0 auto 46px}
.chip{white-space:nowrap;font-weight:800;font-size:44px;padding:8px 24px;border-radius:var(--r-chip)}
.chip--navy{background:var(--navy);color:#fff}
.chip--white{background:#fff;color:var(--navy);box-shadow:0 8px 20px rgba(1,37,79,.12)}
.sticker .a{align-self:flex-start;transform:rotate(-3deg)}
.sticker .b{align-self:flex-end;margin-top:-2px;transform:rotate(2.5deg);position:relative;z-index:2}
```
⚠️ Обязательно `white-space:nowrap` + `width:fit-content`, иначе длинный текст («ВСЕ ДЛЯ СОВРЕМЕННОГО») переносится и вылезает за плашку. Сочетания цветов плашек смотреть по каждому экрану (бывает белый+navy, navy+navy, белый+белый на тёмном фоне).

### 3.2 Фирменная «S»-лента
Большая волнистая лента (синяя `--blue` или белая) «вытекает» из логотипа и проходит **сквозь секции** фоновым слоем. Рисуется SVG-плавной кривой толстым `stroke` с `stroke-linecap:round`, `position:absolute`, `z-index:0`, контент — `z-index:1`. Есть в блоках: преимущества, фичер «детский сад», о компании, контакты-панель. Не путать с контентом — это декор.

### 3.3 Точечная текстура (dot-grid)
Светлые секции покрыты сеткой точек:
```css
background:var(--sky);
background-image:radial-gradient(#cbe0f7 2px,transparent 2px);
background-size:30px 30px;
```
На тёмных секциях — `rgba(255,255,255,.08)` точки.

### 3.4 Кнопка + круглая стрелка (парный элемент)
Рядом с КАЖДОЙ pill-кнопкой стоит отдельная **круглая** кнопка со стрелкой `→`.
```html
<div class="cta">
  <button class="btn">Перейти в каталог</button>
  <button class="btn-arrow" aria-label="Далее"><svg.../></button>
</div>
```
`.btn`{background:var(--blue);color:#fff;border-radius:var(--r-pill);padding:15px 30px} · `.btn-arrow`{width:52px;height:52px;border-radius:50%;background:var(--blue)}. Состояние нажатия везде — `transform:scale(.97)`.

### 3.5 Синие «бегущие строки» (advantage bar)
Полноширинная полоса `--blue`, пункты белым UPPERCASE 800, между ними — иконка-логотип (рамка-квадрат с «S»). Две штуки на главной (3 и 4 пункта).

### 3.6 Тёмные фото-плитки категорий
Фото + затемнение `linear-gradient(180deg,rgba(1,18,39,.25),rgba(1,18,39,.78))` + белый UPPERCASE заголовок снизу-слева + круглая синяя стрелка снизу-справа.

### 3.7 Логотип
Синий скруглённый квадрат (`--r-chip`) с белой курсивной «S» + текст `CLASS TRADE` (800, `white-space:nowrap`). Та же иконка-«S» — в разделителях синих полос и в подвале.

### 3.8 Единая тень только под фото
`--shadow-photo` применяется **только к фото-рендерам**. Карточки — мягкая `--shadow-card` или тонкая граница, без тяжёлых теней. Не вешать тени на кнопки/чипы (кроме лёгкой подсветки белых чипов).

---

## 4. Скругления (строго)
плашка-стикер `11px` · фото-плитка `18px` · карточка `22px` · кнопка/инпут/счётчик `pill` · фото-рендеры `16px`.

## 5. Состояния и анимации
- Нажатие любой кнопки: `scale(.97)`, переход ~120ms.
- Слайдеры (hero `01/03`, фичер `01/04`) — стрелки ‹ ›, активная пагинация.
- Аккордеон «Сферы внедрения»: активный пункт синий + раскрывает подпись и меняет фото слева.
- Ссылки — подчёркивание по hover. Без «прыгающих» эффектов.

## 6. Контент и копирайт
- Всё на русском, тон деловой, заголовки UPPERCASE. **Без эмодзи.**
- Заглушки в макете (LOGO партнёров, «+7 (4822) 00-00-00», «ГОРОД, УЛИЦА, ДОМ», «НАЗВАНИЕ НОВОСТИ») — оставить как есть до получения реальных данных от заказчика, не выдумывать.

## 7. Ассеты
Фото-рендеры лежат в `assets/` (классы, кабинеты, спортзал, детсад) — извлечены из PDF. Логотипы партнёров — заглушки. Иконки (поиск, сердце, корзина, гамбургер, стрелки, чекбокс) — тонкие линейные SVG (stroke ~2px), монохром: `--ink`/белый, синий только когда сам элемент интерактивен.

## 8. Структура главной (порядок секций)
1. Шапка (navy) → 2. Hero (navy+точки, H1+стикеры, 2 фото под наклоном, слайдер 01/03) → 3. Преимущества (3 карточки + S-лента) → 4. Синяя полоса ×1 → 5. Категории (2+3 фото-плитки) → 6. Фичер «Магазин уникальных решений» (синяя панель + фото + S-лента, слайдер 01/04) → 7. Сферы внедрения (фото + аккордеон 4 пункта) → 8. О компании (текст + 2 фото + S-лента) → 9. Синяя полоса ×2 → 10. Партнёры (10 плиток LOGO) → 11. Новости (3 карточки) → 12. Контакты (форма + синяя панель + S-лента, на navy) → 13. Подвал (navy).

## 9. Чего НЕ делать
- ❌ Не выравнивать стикер-заголовки «в одну строку» — они диагональные.
- ❌ Не добавлять новые цвета/градиенты/тени помимо описанных.
- ❌ Не менять копирайт и заглушки на свой текст.
- ❌ Не применять сторонние UI-киты/«минимализм» — стиль строго по макету CLASS TRADE.
- ❌ Не убирать S-ленту и точечный фон ради «чистоты».
